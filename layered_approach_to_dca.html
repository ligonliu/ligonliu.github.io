My two-layer approach to optimal wireless Channel Allocation problem

fractional channel, adaptive noise toleration, minimal then decreasing marginal utility - no title specified

 

Comments from 2022: performance enhancement of embedded processors and ubiquitous cellular connectivity have eliminated the need of decentralized channel contention, because the optimal channel assignments can be computed on a single head node or a cloud server. The interference matrix, bandwidth-SINR abstraction and bandwidth utility models are still valid. I archive my CFL algorithm variants in my little museum at repo

I classify the current investigated problems in dynamic wireless channel allocation into two relatively independent layers.

In the bottom is the topological layer, which abstracts physical constraints of channel sharing; in the middle exists the utility (or pricing) layer, which abstracts users’ benefit in using the channels.

Abstract

The following models, problems and approaches are proposed:

Topological Layer:

Fractional channel: Wireless channel is divisible to an extent.

Interference-tolerant channel: Channel’s bandwidth can adapt to SINR by adjusting encoding

Send/Receive asymmetric channel: Send and receive is considered separately

Utility Layer:

Concave utility function

Minimal bandwidth & additional utility bandwidth Dualism

Minimal bandwidth channel allocation, based on fractional graph coloring algorithm

Additional utility channel allocation, based on Maximum Marginal Utility Independent Set

Topological Layer

Different models exist in the topological layer. The most widely researched is the simplest model, which can be referred as “0-1 interference”. This model assumes:

  • Two nodes are either interfering or not interfering with each other, if they both transmit or receive in the same channel.  

  • The number of atomic, non-dividable channels is limited. 

The 0-1 interference can be modeled with simple graph coloring and maximum independent set problem.

In the real world, some exceptions happen beyond 0-1 interference, like the hidden node and exposed node problem, but I am not trying to amend the model for these exceptions.

 

 

Fractional channel model(improvement over non-dividable channel)

In the real world, efficient methods to divide and share channel exists in the form of FD,TD and CD MA. Multi-subcarrier baseband and RF is prevalent among communication devices ever since the DSL modem, such that a device can use multiple fractional channels simultaneously.

The benefit of fractional channel is very clear: it allows using less total spectrum to provide same bandwidth for users. Also, as a relaxation, fractional graph coloring problem is easier to solve than integer graph coloring problem.

 

 

Consider example of the pentagon graph:

If 1 non-dividable 10MHz channel is required by each node, the interference graph need 30MHz spectrum to fulfill the 5 nodes; However, if channels can be fractured by 2, Which means, each node need two 5MHz channels, the interference graph only need 25MHz spectrum.

The difference is in the latter graph, for at least one node, the 10MHz are not continuous. (two separate 5MHz channels)

Note:

The Shannon-Hartley theorem has interesting indications that there is an optimal balance between spectrum availability and SINR. This balance is again ignored by graph coloring modeling. SINR decrease by 3db when interference energy is doubled. In a high SINR system, the bandwidth increase brought by SINR doubling is marginal where in a low SINR system the bandwidth increase brought by SINR doubling can be very significant. SINR is a marginal benefit decreasing resource here.

 

Send/Receive asymmetry

In wireless network, different interference patterns exists for sending and receiving, assuming sending and receiving is on different channels. It creates hidden node and exposed node problem, when graph coloring model treat send interference and receive interference in the same, unit graph way.

The way to exploit this asymmetry is creating two different entities: send (uplink) and receive (downlink) bandwidth. The attenuation matrix remains the same. The utility function (mentioned below) would become a two variable function util(bs ,br), bs  is send bandwidth, br  is receive bandwidth) instead of single variable function util(b).

Utility Layer

Concave utility function

The economics definition of utility reflects node’s incentive to use certain bandwidth, or the rent a node would like to pay for a spectrum used in a certain period of time. An example is there are people who need only 0.1 channel for voice call, and people who need 3.2 channels for watching online video. Graph coloring model has omitted this utility (and thus demand) curve.

“Concave utility function” is an assumption that can minimize computation complexity.  It has the same meaning with “marginal utility of bandwidth decreases” in economics term. The analysis will be covered later in this section.

 

“Concave from minimal up” utility function

In reality, a certain application usually needs a minimal bandwidth to be able to do things, like streaming music need at least 16kbps, online video need at least 256kbps etc. If the bandwidth is lower than the minimum, the utility is nearly zero.

Minimal bandwidth channel allocation, based on fractional graph coloring algorithm

This is already solved by a decentralized iterative randomized algorithm called MultiCFL, which I invented based on the idea of CFL algorithm.

MultiCFL is for fractionally coloring a graph. Each node just sense surrounding interference and change its channel accordingly, there is no need of intercommunication among nodes.

We still need to investigate MultiCFL’s speed (sensing cycles) of convergence, as each MultiCFL cycle need a sensing operation, which can be slow and energy costly.

There exist other centralized algorithms, which is another direction to explore.

 

Algorithm for Extra Utility channel allocation, based on Maximum Weight Independent Set

MultiCFL has only solved the welfare bandwidth allocation problem. However, it also helps solve the extra utility bandwidth problem by providing an approximate fractional coloring of conflict graph.

Fractional graph coloring can also be solved in a greedy manner by incrementally coloring the graph’s maximum independent set with a new color.

In our case with utility, we are trying to find a maximum weight (each node’s weight is its marginal utility) independent set. Finding maximum independent set on unit disk graph is polynomial time solvable.[5] We want to see if the efficient algorithm can be extended into the case every node has a weight. Also, we want to see if there is efficient decentralized algorithm for finding maximum weight independent set on more general graphs.

 

Conclusion of Utility Layer

The proposed utility function is 0 when bandwidth is less than minimal, and concave up from a minimal bandwidth. Allocation of minimal bandwidth can be solved with fractional graph coloring algorithm, while efficient allocation of additional bandwidth can be modeled with maximum weight independent set problem.

Remember, all those analyses are based on the basic 0-1 unidirectional interference model. The analysis should be extended to the models proposed in the topological part of this article.
